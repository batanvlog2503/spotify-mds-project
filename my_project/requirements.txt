⚠️ Lưu ý quan trọng
Prophet cần thêm dependency hệ thống trước khi pip install:
bash# Ubuntu/Debian
sudo apt-get install python3-dev libpython3-dev

# MacOS
brew install gcc
PySpark cần Java 11+:
bash# Kiểm tra
java -version

# Cài nếu chưa có (Ubuntu)
sudo apt install openjdk-11-jdk
dbt-snowflake và snowflake-connector hay conflict version — nên cài trong virtual environment riêng:
bashpython -m venv venv
source venv/bin/activate   # MacOS/Linux
venv\Scripts\activate      # Windows
pip install -r requirements.txt
